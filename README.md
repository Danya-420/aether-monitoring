# Aether.OS 🖥️

A minimalist activity tracking dashboard for conscious digital well-being. Built with Electron, React, and Vite.

## ✨ Features

- **📊 Daily Flow Monitor** - Visualize your hourly activity patterns with smooth SVG curves
- **⌨️ Input Pulse** - Track typing speed (WPM) with weekly comparisons
-   **7-Day Heatmap** - GitHub-style activity contribution grid
- **📦 App Inventory** - See which applications consume your time
- **🔒 Privacy-First** - Incognito mode, auto data retention (30 days), local-only storage
- **⚡ Background Mode** - Runs silently in system tray, auto-starts on login

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Windows, macOS, or Linux (for packaged build)

### Installation

```bash
# Clone the repository
git clone https://github.com/aether-team/aether.git
cd aether

# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build

# Package as Windows installer
npm run package
```
